Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir

  • Parágrafo Use a tag (p e tag /p)
  • Titulos Use as tag de h1 a h6
  • Imagens utilizamos a tag IMG
  • Abreviaçõs utilizamos a tag ABBR e /Abbr
  • Código fomte utilizamos a tag CODE e /CODE
  • Lista com demarcadores usamos a tag ol junto com tag LI
  • Lista com demarcadores usamos a tag OL e /OL
